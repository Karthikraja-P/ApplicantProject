  Q7  Type : Shape Pattern Matrix
      Question : 3x3 grid showing shapes (square, circle, pentagon) with
                 increasing fill (outline, half-filled, fully-filled).

  Q8  Type : Number Matrix (Row Sum)
      Question : Find the missing number:
                   4   3   7
                   3   6   9
                   2   9   ?
                 A. 9   B. 7   C. 11   D. 6
      Answer   : C. 11  (each row: col1 + col2 = col3)
      Source   : Sample PDF

  Q9  Type : Number Matrix (Row Product)
      Question : Find the missing number:
                   5   3  15
                   3   3   9
                   4   2   ?
                 A. 6   B. 12   C. 2   D. 8
      Answer   : D. 8  (each row: col1 x col2 = col3)
      Source   : Sample PDF

  Q10 Type : Number Matrix (Decreasing Differences)
      Question : Find the missing number:

                  37  33  28
                  22  15   ?

  Q17 Type : Food Chain Cascade
      Question : In a food chain:  Grass --> Rabbit --> Fox
                 All Rabbits are suddenly removed. What is the most likely
                 outcome?
                 A. Fox population increases; Grass decreases.
                 B. Fox population decreases; Grass increases.
                 C. Fox population increases; Grass increases.
                 D. No change in either Fox or Grass populations.
      Answer   : B  (Fox loses food; Grass expands without grazing)
      Source   : SebastienEveno / McKinsey Solve

  Q18 Type : Constraint Logic (Species Extinction)
      Question : 4 predators share 2 prey species.
                   Species A  eats X only
                   Species B  eats Y only
                   Species C  eats both X and Y
                   Species D  eats both X and Y
                 If Species X goes extinct, which predators survive?
                 A. A, B, C and D all survive.
                 B. B, C and D survive; A goes extinct.
                 C. Only B survives.
                 D. None survive.
      Answer   : B
      Source   : SebastienEveno / McKinsey Solve

  Q19 Type : Ecosystem Building (Sustainable Chain)
      Question : Form a valid 3-species food chain from:
                 Eagle, Trout, Mayfly, Deer
                   Eagle eats Trout and Deer
                   Trout eats Mayfly
                   Deer  eats Grass (external)
                   Mayfly eats Algae (external)
                 Which trio forms a complete valid chain?
                 A. Eagle --> Deer  --> Mayfly
                 B. Eagle --> Trout --> Mayfly
                 C. Deer  --> Eagle --> Trout
                 D. Eagle --> Deer  --> Trout
      Answer   : B  (Eagle eats Trout, Trout eats Mayfly, Mayfly eats Algae)
      Source   : SebastienEveno / McKinsey Solve
